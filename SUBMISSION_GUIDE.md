# SUBMISSION_GUIDE

## What to submit
- A GitHub repository containing this code and documentation.

## Before you push
- Confirm there are **no API keys** in:
  - code
  - notebook outputs
  - README/docs
  - git history

## How to run (include in email/message)
- Local:
```bash
pip install -r requirements.txt
export OPENAI_API_KEY="..."
python main.py
```
- Offline:
```bash
export USE_MOCK=true
python main.py
```
